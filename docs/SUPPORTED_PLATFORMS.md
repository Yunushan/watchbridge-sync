# Supported Platforms

| Platform | Status | Notes |
|---|---|---|
| Web | First-class | Browser/PWA UI. |
| Windows | First-class | Desktop wrapper, CLI, and API server. |
| Windows Server | First-class | API server, workers, and CLI. |
| Linux | First-class | Desktop wrapper, CLI, and API server. |
| BSD | Supported for server/CLI/web | Native desktop depends on available WebView/Electron/Tauri runtime. |
| Android | Planned first-class | Capacitor or React Native wrapper. |
| iOS | Planned first-class | Capacitor or React Native wrapper. |
| macOS | Planned first-class | Desktop wrapper and CLI. |

The core logic is TypeScript and can run anywhere modern Node.js runs. The web UI can be served by any static web server.
